Wisconsin Cancer Rate Study


Technologies Used: ArcPy | TKinter


The objective for project #1 in Geography 777 was to create an application that allows the user to explore the relationship between nitrate values in wells and cancer rates at the census tract level in Wisconsin. This application was developed using a python solution utilizing Arcpy and Tkinter. The GUI was built using Tkinter which allows even a non-GIS professional to create and explore maps comparing this relationship. The GUI will require the user to input a k value for the decay coefficient that will be used for the IDW interpolation. Once the user submits a k value greater than one, a separate analysis script utilizing Arcpy tools will run outputting two maps, one of the IDW interpolation of the nitrate values, and the other a map of the linear regression results.
